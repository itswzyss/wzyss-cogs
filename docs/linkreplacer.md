# LinkReplacer

**Short description:** Replace links with configured alternatives.

## Description

Automatically replaces configured links with alternatives. For example, replace X.com links with fixupx.com links.

## Install

```
[p]cog install wzyss-cogs linkreplacer
```

## Requirements

None.

## Tags

links, utility, moderation

See `[p]help linkreplacer` to see available commands.
