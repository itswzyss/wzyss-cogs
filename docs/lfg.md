# LFG

**Short description:** Looking for Group: register availability per game, see who's available, notify via DM, request games.

## Description

Guild-scoped LFG: masterlist of games, per-user available/unavailable per game, view who is available, notify (DM) opted-in users, and request new games for admin approval.

## Install

```
[p]cog install wzyss-cogs lfg
```

## Requirements

None.

## Tags

utility, games, lfg

## Commands

See `[p]help lfg` for commands. Users can set availability per game, see who's available, request games, and opt in to DM notifications when someone notifies for a game.
