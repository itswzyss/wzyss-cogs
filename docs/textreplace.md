# TextReplace

**Short description:** Replace text using Regex.

## Description

Automatically replaces text using Regex. This cog can repost messages with text replaced according to configured regex patterns.

## Install

```
[p]cog install wzyss-cogs textreplace
```

## Requirements

None.

## Tags

text, utility, moderation, regex

## Commands

See `[p]help textreplace` to see available commands.
