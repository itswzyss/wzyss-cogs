# Counting

**Short description:** Count upwards in channels with optional math expressions.

## Description

Allows members to count upwards in configured channels. Members can use numbers or math expressions (e.g., '1+1' for 2). Starts at 1 and can have an optional goal. Prevents the same user from counting twice in a row.

## Install

```
[p]cog install wzyss-cogs counting
```

## Requirements

None.
