# Clear

**Short description:** Clear/purge messages in a channel (by count, after message, between messages, or by user).

## Description

Allows mods and admins to manage messages in a channel: clear a number of messages, clear all messages after a given message, clear messages between two messages, or clear messages from a specific user. Supports both slash and prefix commands.

## Install

```
[p]cog install wzyss-cogs clear
```

## Requirements

None.

## Tags

moderation, utility, messages

## Commands

See `[p]help clear` for commands. Requires Manage Messages (or mod) permission.
