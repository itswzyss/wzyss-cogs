# RockstarNewswire

**Short description:** Track and post Rockstar Games newswire updates.

## Description

Automatically tracks Rockstar Games newswire feed and posts updates to configured channels. Supports multiple news types including GTA V, GTA VI, RDR2, music, fanart, and more.

## Install

```
[p]cog install wzyss-cogs rockstarnewswire
```

## Requirements

- aiohttp
- playwright

## Tags

news, rockstar, gta, rdr2, notifications, utility
