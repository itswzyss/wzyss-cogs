# AutoVC

**Short description:** Automatically create voice channels when members join source VCs.

## Description

Automatically creates voice channels when members join configured source voice channels. Supports three types: Public (anyone can join), Personal (owner-controlled, visible), and Private (owner-controlled, hidden). Owners control their VC via bot commands and an optional VC panel (embed with buttons); no Discord role is assigned, avoiding 2FA-for-mods. Features automatic cleanup and owner claiming.

## Install

```
[p]cog install wzyss-cogs autovc
```

## Requirements

None.
