# Giveaway

**Short description:** Reaction-based giveaways with optional claim system.

## Description

Run giveaways with reaction-based entry, interactive or command-based setup, and management (reroll, end, edit, cancel). Optional configurable claim window with automatic reroll if not claimed.

## Install

```
[p]cog install wzyss-cogs giveaway
```

## Requirements

None.

## Tags

giveaway, utility, reactions
