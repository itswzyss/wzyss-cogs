# BoosterRole

**Short description:** Track booster custom roles and remove them when a user stops boosting.

## Description

When a user loses the Server Booster role, identifies their custom role (manual or auto: single-occupant, name pattern, position below booster), removes that role from the user, and logs to a channel with optional role ping.

## Install

```
[p]cog install wzyss-cogs boosterrole
```

## Requirements

None.
